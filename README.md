#kBot
kBot is an IRC bot written in Python.

##Features:
- Google search
- Definition search
- Draw images in ascii art
- Code evaluation
- Quote users
- Decide (randomly choose between two options)
- Link reader
- Vim-like sub command
- Friendly greetings and responses
